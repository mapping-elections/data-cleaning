Atlas of Historical County Boundaries: Comprehensive Database

 

 

READ ME

 

The comprehensive database for each state is supplied as a tab-delimited text file which may be imported into a database or spreadsheet program.

The first row of the database contains the field names and full descriptions of each field are supplied below.

 

FIELD NAME

DATA TYPE

DESCRIPTION

 

 

 

FIPS

Text – 5 characters

Federal Information Processing Standard codes used by the U.S. Census Bureau. Counties and equivalents (i.e. extinct and non-county areas) not included in the published FIPS codes have not been assigned codes by this project.

 

 

 

STATE

Text – 2 characters

Modern state that has jurisdiction of the county or equivalent. State name is presented in the standard 2-letter abbreviation used by the U.S. Postal Service.

 

 

 

NAME

Text – 50 characters

Name of the county or equivalent (i.e. an Independent City, a disputed area, or a non-county area.

 

 

 

ID

Text – 50 characters

Unique identifier code for counties and equivalents.

 

 

 

VERSION

Text - 6 characters

Version number tracks sequential and chronological changes in the county name or configuration.

 

 

 

START_DATE

Date/Time

Starting date for a particular county version or event. Date is presented in “dd mmm yyyy” format (i.e. 02 Apr 1850)

 

 

 

END_DATE

Date/Time

Ending date for a particular county version or event. Date is presented in “dd mmm yyyy” format (i.e. 02 Apr 1850)

 

 

 

CREATION_OR_CHANGE

Memo

Full description of the creation, change, or other historical event.

 

 

 

CITATION

Memo

Source citation for the creation, change, or other historical event.

 

 

 

START_N

Number – long integer

Start Date presented as a numeric field, in “yyyymmdd” format (i.e. 18500402).

 

 

 

END_N

Number – long integer

End Date presented as a numeric field, in “yyyymmdd” format (i.e. 18500402).

 

 

 

DATASET

Text - 25 characters

Identifier for the full set of data (maps, database, and supplemental texts) included in the downloadable file.
